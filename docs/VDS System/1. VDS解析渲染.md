## **阶段1. 配置编辑（vds-configurator.js）**



用户打开 VDS 编辑器 (/LowCode/Index)

&#x20;      │

&#x20;      ▼

VdsConfigurator.init()

&#x20;      │  绑定 Bootstrap Modal、监听 Tab 切换

&#x20;      │

&#x20;      ▼

用户点击「添加」或「配置」按钮

&#x20;      │

&#x20;      ├── 新建 → VdsConfigurator.create()     重置表单 + 打开 Modal

&#x20;      └── 编辑 → VdsConfigurator.edit(id)     POST /LowCode/GetPage 加载已有配置

&#x20;                                                → loadPageData(page) 填充表单 + fields 数组

&#x20;      │

&#x20;      ▼

用户在 Modal 中操作（3个Tab）

&#x20;      │

&#x20;      ├── 【基础配置】name / title / description / apiUrl / pageSize / orderRules ...

&#x20;      ├── 【字段配置】addField → editField → saveField  → 操作 this.fields\[] 数组

&#x20;      │      每个字段包含: { name, title, type, searchedByKeywords, enumValus, showPart ... }

&#x20;      └── 【接口配置】modalSettings（增/改接口）、customActions

## 阶段2. 保存入库（vds-configurator.js + LowCodeController）

用户点击「保存」

&#x20;      │

&#x20;      ▼

VdsConfigurator.save()

&#x20;      │

&#x20;      ▼

buildVdsConfig()  从表单收集配置，组装为 JSON 对象

&#x20;      │  {

&#x20;      │    apiUrl, pageSize, idFieldName,

&#x20;      │    ths: this.fields,          ← 字段列表（含 searchedByKeywords 标记）

&#x20;      │    modalSettings: {...},

&#x20;      │    orderRules: \[...],

&#x20;      │    customActions: \[...]

&#x20;      │  }

&#x20;      │

&#x20;      ▼

FormData 打包 → POST /LowCode/AddPage 或 /LowCode/UpdatePage

&#x20;      │  字段: name, title, description, orderNo, isEnabled, vdsConfig(JSON字符串)

&#x20;      │

&#x20;      ▼

LowCodeController.AddPage / UpdatePage

&#x20;      │  1. 检查 name 唯一性（WHERE name = @name）

&#x20;      │  2. repository.AddAsync(vdsPage) 或 repository.UpdateAsync(vdsPage)

&#x20;      │     → 写入 VdsPages 表，vdsConfig 作为 JSON 字符串存入 VdsConfig 字段

&#x20;      │

&#x20;      ▼

返回成功 → Modal 关闭 → tables\['vdsPageTable'].loadData() 刷新列表





## 阶段3. 点击预览, 请求页面(LowCodeController + Render.cshtml)

用户在列表中点击「预览」

&#x20;      │  onclick="window.open('/LowCode/Render/' + '{{name}}', '\_blank')"

&#x20;      │

&#x20;      ▼

新标签页 → GET /LowCode/Render/{pageName}

&#x20;      │

&#x20;      ▼

LowCodeController.Render(pageName)

&#x20;      │  1. WHERE name = @pageName 查 VdsPages 表

&#x20;      │  2. 不存在 → 404

&#x20;      │  3. IsEnabled=false → 404

&#x20;      │  4. 返回 View("Render", vdsPage)   ← VdsPage 实体作为 Model

&#x20;      │

&#x20;      ▼

Render.cshtml 渲染

&#x20;      │  1. @Html.Raw(Model.VdsConfig)  → JSON 字符串直接输出为 JS 对象

&#x20;      │  2. 补充运行时属性:

&#x20;      │     vdsConfig.tableId = '{name}-table'

&#x20;      │     vdsConfig.tableContainerSelector = '#{name}-container'

&#x20;      │     默认 pageIndex=1, pageSize=10, idFieldName='id'

&#x20;      │

&#x20;      ▼

调用 createTable(vdsConfig)
